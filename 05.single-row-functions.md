## Character Functions

### UPPER
The `UPPER` function returns a character expression with all lowercase character data converted to uppercase.

**Arguments:**
- **character_expression**: The string expression to be converted to uppercase. This can be a constant, variable, or column of either character or binary data.

**Syntax:**
```sql
UPPER ( character_expression )
```

**Examples:**
```sql
-- Example 1: Convert a lowercase string to uppercase
SELECT UPPER('sql server'); 
-- Output: 'SQL SERVER'

-- Example 2: Convert a mixed-case string to uppercase
SELECT UPPER('Hello World'); 
-- Output: 'HELLO WORLD'

-- Example 3: Convert a mixed alphanumeric string to uppercase
SELECT UPPER('MixedCase123'); 
-- Output: 'MIXEDCASE123'

-- Example 4: Convert a numeric string to uppercase (no effect on numbers)
SELECT UPPER('123abc'); 
-- Output: '123ABC'

-- Example 5: Convert a single word
SELECT UPPER('lowercase'); 
-- Output: 'LOWERCASE'
```

### LOWER

**Arguments:**
- **character_expression**: The string expression to be converted to lowercase. This can be a constant, variable, or column of either character or binary data.

```sql
LOWER ( character_expression )
```

```sql
-- Example 5: Convert a single word

```

### LEN
The `LEN` function returns the number of characters in a string expression, excluding trailing spaces.

**Arguments:**
- **string_expression**: The string expression to evaluate. This can be a constant, variable, or column.

**Syntax:**
```sql
LEN ( string_expression )
```

**Examples:**
```sql
-- Example 1: Get the length of a string
SELECT LEN('SQL Server'); 
-- Output: 10

-- Example 2: Get the length of a string with spaces
SELECT LEN('Hello World'); 
-- Output: 11

-- Example 3: Get the length of a string with trailing spaces (spaces ignored)
SELECT LEN('Test123   '); 
-- Output: 7

-- Example 4: Get the length of a numeric string
SELECT LEN('12345'); 
-- Output: 5

-- Example 5: Get the length of an empty string
SELECT LEN(''); 
-- Output: 0
```

### CONCAT
The `CONCAT` function returns a string that is the result of concatenating two or more string values.

**Arguments:**
- **string_value1, string_value2, ...**: The string values to concatenate. You can specify any number of arguments. The result of the concatenation is a single string.

**Syntax:**
```sql
CONCAT ( string_value1, string_value2 [, string_valueN ] )
```

**Examples:**
```sql
-- Example 1: Concatenate two strings
SELECT CONCAT('SQL', ' Server'); 
-- Output: 'SQL Server'

-- Example 2: Concatenate multiple strings
SELECT CONCAT('Hello', ' ', 'World'); 
-- Output: 'Hello World'

-- Example 3: Concatenate strings with numbers
SELECT CONCAT('Year ', 2024); 
-- Output: 'Year 2024'

-- Example 4: Concatenate first and last names
SELECT CONCAT('John', ' ', 'Doe'); 
-- Output: 'John Doe'

-- Example 5: Concatenate a string with a NULL value
SELECT CONCAT('SQL', NULL, 'Server'); 
-- Output: 'SQLServer'
```

### LEFT
The `LEFT` function returns the left part of a character string with the specified number of characters.

**Arguments:**
- **character_expression**: The string expression from which to extract the left part.
- **integer_expression**: The number of characters to return from the left of the string. Must be a positive integer.

**Syntax:**
```sql
LEFT ( character_expression , integer_expression )
```

**Examples:**
```sql
-- Example 1: Get the first 3 characters of a string
SELECT LEFT('SQL Server', 3); 
-- Output: 'SQL'

-- Example 2: Get the first 5 characters of a string
SELECT LEFT('Hello World', 5); 
-- Output: 'Hello'

-- Example 3: Get the first character of a string
SELECT LEFT('abcdef', 1); 
-- Output: 'a'

-- Example 4: Get the first 2 characters of a numeric string
SELECT LEFT('12345', 2); 
-- Output: '12'

-- Example 5: Get the first 4 characters of a string with spaces
SELECT LEFT(' Data Science', 4); 
-- Output: ' Dat'
```

### SUBSTRING
The `SUBSTRING` function returns part of a character, binary, text, or image expression.

**Arguments:**
- **expression**: The string expression from which to extract the substring.
- **start**: The starting position of the substring. This is a positive integer.
- **length**: The number of characters to return. Must be a positive integer.

**Syntax:**
```sql
SUBSTRING ( expression , start , length )
```

**Examples:**
```sql
-- Example 1: Extract the first 3 characters from a string
SELECT SUBSTRING('SQL Server', 1, 3); 
-- Output: 'SQL'

-- Example 2: Extract 5 characters starting from the 7th position
SELECT SUBSTRING('Hello World', 7, 5); 
-- Output: 'World'

-- Example 3: Extract 3 characters starting from the 2nd position
SELECT SUBSTRING('12345', 2, 3); 
-- Output: '234'

-- Example 4: Extract 2 characters starting from the 3rd

 position
SELECT SUBSTRING('abcdef', 3, 2); 
-- Output: 'cd'

-- Example 5: Extract 7 characters starting from the 6th position
SELECT SUBSTRING('Data Science', 6, 7); 
-- Output: 'Science'
```

### CHARINDEX
The `CHARINDEX` function returns the starting position of the specified expression in a character string. If the string is not found, it returns 0.

**Arguments:**
- **expressionToFind**: The substring to search for in the expression.
- **expressionToSearch**: The string expression to search within.
- **start_location** (optional): The starting position to begin the search. This is an integer. If omitted, the search starts at the beginning of the expression.

**Syntax:**
```sql
CHARINDEX ( expressionToFind , expressionToSearch [ , start_location ] )
```

**Examples:**
```sql
-- Example 1: Find the position of a substring
SELECT CHARINDEX('Server', 'SQL Server'); 
-- Output: 5

-- Example 2: Find the position of a substring starting from a specific position
SELECT CHARINDEX('l', 'Hello World', 4); 
-- Output: 10

-- Example 3: Find a non-existing substring (returns 0)
SELECT CHARINDEX('XYZ', 'SQL Server'); 
-- Output: 0

-- Example 4: Find the position of the first occurrence of a character
SELECT CHARINDEX('o', 'Hello World'); 
-- Output: 5

-- Example 5: Find the position of a space in a sentence
SELECT CHARINDEX(' ', 'Hello World'); 
-- Output: 6
```

### TRIM
The `TRIM` function removes the specified characters from the start and end of a string. By default, it removes spaces.

**Arguments:**
- **characters** (optional): The characters to remove from the string. If not specified, spaces are removed.
- **string_expression**: The string expression to trim.

**Syntax:**
```sql
TRIM ( [ characters FROM ] string_expression )
```

**Examples:**
```sql
-- Example 1: Remove leading and trailing spaces
SELECT TRIM('   SQL Server   '); 
-- Output: 'SQL Server'

-- Example 2: Remove specific characters from both ends of the string
SELECT TRIM('x' FROM 'xxSQL Serverxx'); 
-- Output: 'SQL Server'

-- Example 3: Trim leading and trailing tabs
SELECT TRIM(CHAR(9) FROM CHAR(9) + 'SQL Server' + CHAR(9)); 
-- Output: 'SQL Server'

-- Example 4: Remove specific characters from both ends of the string
SELECT TRIM('ab' FROM 'abSQL Serverba'); 
-- Output: 'SQL Server'

-- Example 5: Trim leading and trailing newline characters
SELECT TRIM(CHAR(10) FROM CHAR(10) + 'SQL Server' + CHAR(10)); 
-- Output: 'SQL Server'
```

### REPLACE
The `REPLACE` function replaces all occurrences of a specified substring within a string, with another substring.

**Arguments:**
- **string_expression**: The string expression in which the search and replace operation occurs.
- **string_pattern**: The substring to search for in the string.
- **string_replacement**: The substring to replace the search pattern with.

**Syntax:**
```sql
REPLACE ( string_expression , string_pattern , string_replacement )
```

**Examples:**
```sql
-- Example 1: Replace all occurrences of a substring
SELECT REPLACE('SQL Server 2019', '2019', '2022'); 
-- Output: 'SQL Server 2022'

-- Example 2: Replace spaces with hyphens
SELECT REPLACE('Hello World', ' ', '-'); 
-- Output: 'Hello-World'

-- Example 3: Replace a character in a numeric string
SELECT REPLACE('12345', '2', '8'); 
-- Output: '18345'

-- Example 4: Replace a substring in a mixed string
SELECT REPLACE('abcdef', 'cd', 'xy'); 
-- Output: 'abxyef'

-- Example 5: Replace multiple characters
SELECT REPLACE('Data Science', ' ', ''); 
-- Output: 'DataScience'
```

### TRANSLATE
The `TRANSLATE` function returns the string provided as a first argument with all occurrences of characters from the second argument replaced by the corresponding characters in the third argument.

**Arguments:**
- **inputString**: The string to translate.
- **characters**: The characters to search for in the input string.
- **translations**: The characters to replace the search characters with.

**Syntax:**
```sql
TRANSLATE ( inputString , characters , translations )
```

**Examples:**
```sql
-- Example 1: Translate characters in a string
SELECT TRANSLATE('SQL Server', 'SQ', 'AB'); 
-- Output: 'ABL Server'

-- Example 2: Translate vowels to numbers
SELECT TRANSLATE('Hello World', 'aeiou', '12345'); 
-- Output: 'H2ll4 W4rld'

-- Example 3: Replace multiple characters in a string
SELECT TRANSLATE('abcdef', 'abc', 'xyz'); 
-- Output: 'xyzdef'

-- Example 4: Replace spaces and hyphens
SELECT TRANSLATE('Hello-World', ' -', '_|'); 
-- Output: 'Hello_World'

-- Example 5: Translate numbers to letters
SELECT TRANSLATE('12345', '12345', 'ABCDE'); 
-- Output: 'ABCDE'
```

## Numeric Functions

### ABS
The `ABS` function returns the absolute value of the specified numeric expression.

**Arguments:**
- **numeric_expression**: The numeric expression for which to find the absolute value. This can be an integer, decimal, float, or any other numeric data type.

**Syntax:**
```sql
ABS ( numeric_expression )
```

**Examples:**
```sql
-- Example 1: Get the absolute value of a positive number
SELECT ABS(10); 
-- Output: 10

-- Example 2: Get the absolute value of a negative number
SELECT ABS(-10); 
-- Output: 10

-- Example 3: Get the absolute value of a decimal number
SELECT ABS(-10.5); 
-- Output: 10.5

-- Example 4: Get the absolute value

 of a floating-point number
SELECT ABS(-3.14159); 
-- Output: 3.14159

-- Example 5: Get the absolute value of a calculation
SELECT ABS(-20 + 5); 
-- Output: 15
```

### POWER
The `POWER` function returns the value of the specified expression raised to the specified power.

**Arguments:**
- **numeric_expression**: The base number to be raised to the power.
- **power**: The exponent to raise the base number to. This is a positive or negative integer.

**Syntax:**
```sql
POWER ( numeric_expression , power )
```

**Examples:**
```sql
-- Example 1: Raise 2 to the power of 3
SELECT POWER(2, 3); 
-- Output: 8

-- Example 2: Raise 10 to the power of 4
SELECT POWER(10, 4); 
-- Output: 10000

-- Example 3: Raise 5 to the power of 0
SELECT POWER(5, 0); 
-- Output: 1

-- Example 4: Raise 7 to the power of -2 (fractional result)
SELECT POWER(7, -2); 
-- Output: 0.0204

-- Example 5: Raise a decimal number to a power
SELECT POWER(2.5, 3); 
-- Output: 15.625
```

- **float_expression**: The number for which to find the square root. This must be a positive number.

```sql
SQRT ( float_expression )
```

```sql
SELECT SQRT(16);
```

### ROUND
The `ROUND` function returns a numeric value, rounded to the specified length or precision.

**Arguments:**
- **numeric_expression**: The numeric value to round.
- **length**: The precision to which the numeric expression is to be rounded.
- **function** (optional): A value of 0 (default) rounds to the nearest number. A value of 1 truncates the number.

**Syntax:**
```sql
ROUND ( numeric_expression , length [, function ] )
```

**Examples:**
```sql
-- Example 1: Round a number to 2 decimal places
SELECT ROUND(123.4567, 2); 
-- Output: 123.46

-- Example 2: Round a number to the nearest whole number
SELECT ROUND(123.4567, 0); 
-- Output: 123

-- Example 3: Round a number to 1 decimal place
SELECT ROUND(123.4567, 1); 
-- Output: 123.5

-- Example 4: Round a negative number to 2 decimal places
SELECT ROUND(-123.4567, 2); 
-- Output: -123.46

-- Example 5: Truncate a number to 2 decimal places
SELECT ROUND(123.4567, 2, 1); 
-- Output: 123.45
```

## Date Functions

### GETDATE()
The `GETDATE()` function returns the current database system date and time, without the database time zone offset.

**Arguments:** 
- None

**Syntax:**
```sql
GETDATE()
```

**Examples:**
```sql
-- Example 1: Get the current date and time
SELECT GETDATE(); 
-- Output: 2024-08-31 15:45:12.123 (varies)

-- Example 2: Use GETDATE() in a table insertion
INSERT INTO Orders (OrderDate) VALUES (GETDATE()); 

-- Example 3: Use GETDATE() in a calculation
SELECT GETDATE() + 1; 
-- Output: 2024-09-01 15:45:12.123 (varies)

-- Example 4: Compare a column value with the current date
SELECT * FROM Orders WHERE OrderDate > GETDATE() - 7;

-- Example 5: Get the year from the current date
SELECT YEAR(GETDATE()); 
-- Output: 2024
```


### EOMONTH
The `EOMONTH` function returns the last day of the month that contains the specified date, with an optional offset.

**Arguments:**
- **start_date**: The date for which to find the end of the month.
- **month_to_add** (optional): The number of months to add to the start date before calculating the end of the month.

**Syntax:**
```sql
EOMONTH ( start_date [, month_to_add ] )
```

**Examples:**
```sql
-- Example 1: Get the end of the month for a specific date
SELECT EOMONTH('2024-08-15'); 
-- Output: 2024-08-31

-- Example 2: Get the end of the current month
SELECT EOMONTH(GETDATE()); 
-- Output: 2024-08-31

-- Example 3: Get the end of the month with an offset
SELECT EOMONTH('2024-08-15', 2); 
-- Output: 2024-10-31

-- Example 4: Get the end of the month for a date in the past
SELECT EOMONTH('2024-08-15', -2); 
-- Output: 2024-06-30

-- Example 5: Get the end of the month for a leap year date
SELECT EOMONTH('2024-02-15'); 
-- Output: 2024-02-29
```

The `DATETRUNC` function returns the date truncated to the specified part.

**Arguments:**

- **date_expression**: The date expression to truncate.

**Syntax:**
```sql
DATETRUNC ( datepart , date_expression )
```

**Examples:**
```sql
-- Example 1: Truncate the date to the beginning of the year
SELECT DATETRUNC(year, '2024-08-31'); 
-- Output: 2024-01-01

-- Example 2: Truncate the date to the beginning of the month
SELECT DATETRUNC(month, '2024-08-31'); 
-- Output: 2024-08-01

-- Example 3: Truncate the date to the beginning of the day
SELECT DATETRUNC(day, '2024-08-31 15:45:12'); 
-- Output: 2024-08-31 00:00:00

-- Example 4: Truncate the date to the beginning of the quarter
SELECT DATETRUNC(quarter, '2024-08-31'); 
-- Output: 2024-07-01

-- Example 5: Truncate the date to the beginning of the week
SELECT DATETRUNC(week, '2024-08-31'); 
-- Output: 2024-08-25
```

### DATEDIFF
The `DATEDIFF` function returns the count of the specified datepart boundaries crossed between the specified start date and end date.

**Arguments:**
- **datepart**: The part of the date to measure the difference (e.g., year, month, day).
- **start_date**: The starting date.
- **end_date**: The ending date.

**Syntax:**
```sql
DATEDIFF ( datepart , start_date , end_date )
```

**Examples:**
```sql
-- Example 1: Calculate the difference in days
SELECT DATEDIFF(day, '2024-08-01', '2024-08-31'); 
-- Output: 30

-- Example 2: Calculate the difference in months
SELECT DATEDIFF(month, '2024-01-01', '2024-08-31'); 
-- Output: 7

-- Example 3: Calculate the difference in years
SELECT DATEDIFF(year, '2020-01-01', '2024-08-31'); 
-- Output: 4

-- Example 4: Calculate the difference in hours
SELECT DATEDIFF(hour, '2024-08-31 08:00:00', '2024-08-31 20:00:00'); 
-- Output: 12

-- Example 5: Calculate the difference in seconds
SELECT DATEDIFF(second, '2024-08-31 15:45:12', '2024-08-31 15:46:12'); 
-- Output: 60
```

### DATEADD
The `DATEADD` function returns a new datetime value by adding a specified number of dateparts to the specified date.

**Arguments:**
- **datepart**: The part of the date to add (e.g., year, month, day).
- **number**: The number of dateparts to add.
- **date_expression**: The date to which the addition is applied.

**Syntax:**
```sql
DATEADD ( datepart , number , date_expression )
```

**Examples:**
```sql
-- Example 1: Add 1 year to a date
SELECT DATEADD(year, 1, '2024-08-31'); 
-- Output: 2025-08-31

-- Example 2: Add 2 months to a date
SELECT DATEADD(month, 2, '2024-08-31'); 
-- Output: 2024-10-31

-- Example 3: Add 10 days to a date
SELECT DATEADD(day, 10, '2024-08-31'); 
-- Output: 2024-09-10

-- Example 4: Add 3 hours to a datetime value
SELECT DATEADD(hour, 3, '2024-08-31 15:45:12'); 
-- Output: 2024-08-31 18:45:12

-- Example 5: Subtract 5 minutes from a datetime value
SELECT DATEADD(minute, -5, '2024-08-31 15:45:12'); 
-- Output: 2024-08-31 15:40:12
```

## Data Type Conversion Functions

### CAST and CONVERT
The `CAST` and `CONVERT` functions are used to convert an expression of one data type to another.

**Arguments:**
- **expression**: The expression to be converted.
- **data_type**: The target data type.
- **length** (optional for `CONVERT`): The length of the target data type.

**Syntax:**
```sql
CAST ( expression AS data_type )
CONVERT ( data_type [(length)], expression [, style] )
```

**Examples:**
```sql
-- Example 1: Convert an integer to a varchar using CAST
SELECT CAST(12345 AS VARCHAR(10)); 
-- Output: '12345'

-- Example 2: Convert a datetime to a varchar using CONVERT
SELECT CONVERT(VARCHAR(20), GETDATE(), 120); 
-- Output: '2024-08-31 15:45:12' (varies)

-- Example 3: Convert a float to an integer using CAST
SELECT CAST(123.456 AS INT); 
-- Output: 123

-- Example 4: Convert a varchar to a datetime using CONVERT
SELECT CONVERT(DATETIME, '2024-08-31 15:45:12', 120); 
-- Output: '2024-08-31 15:45:12.000'

-- Example 5: Convert an integer to a binary using CAST
SELECT CAST(123 AS BINARY(4)); 
-- Output: 0x0000007B
```

### ISNULL
The `ISNULL` function replaces `NULL` with the specified replacement value.

**Arguments:**
- **check_expression**: The expression to test for `NULL`.
- **replacement_value**: The value to return if `check_expression` is `NULL`.

**Syntax:**
```sql
ISNULL ( check_expression , replacement_value )
```

**Examples:**
```sql
-- Example 1: Replace NULL with a string
SELECT ISNULL(NULL, 'No Value'); 
-- Output: 'No Value'

-- Example 2: Replace NULL with a number


SELECT ISNULL(NULL, 0); 
-- Output: 0

-- Example 3: Use ISNULL in a SELECT statement
SELECT ISNULL(Discount, 0) FROM Sales;

-- Example 4: Use ISNULL with a calculated field
SELECT ISNULL(Price * Quantity, 0) FROM Orders;

-- Example 5: Replace NULL with a date
SELECT ISNULL(NULL, GETDATE()); 
-- Output: Current date and time
```
